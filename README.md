## 📌 What this app does

agentic-chatops is a desktop app for running ChatOps on Windows. It helps one person manage a large device fleet with a simple flow. The app ties together n8n, GPT-4o, Claude Code, and agentic patterns to help with tasks like:

- checking device status
- sending commands
- handling routine fixes
- routing work between agents
- tracking actions in one place

It is built for a solo operator who needs a clear way to manage many devices without switching between many tools.

## 🧠 How it works

The app uses three layers:

- **Chat layer** for your messages and commands
- **Agent layer** for task handling and planning
- **Workflow layer** for automation and routing

This setup helps the app decide what to do, which tool to use, and how to move each task through the system.

It also follows 21 agentic design patterns. In plain terms, that means the app is built to break work into steps, check results, and pass tasks to the right part of the system.

## 🛠️ Troubleshooting

### The app does not open

- Check that the download finished.
- Try opening it again as the current user.
- Restart Windows and try once more.

### Windows blocks the file

- Open the file properties.
- Check for an Unblock option.
- Allow the file if you trust the release.

### The app cannot connect to services

- Check your internet connection.
- Confirm your API key is correct.
- Check your n8n URL or token.
- Make sure the target service is online.

### Commands do not run as expected

- Start with one small task.
- Check your workflow settings.
- Confirm the device or host name is correct.
- Review logs for the failed step.
